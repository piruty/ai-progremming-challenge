# Image Resizer & Compressor

画像をドラッグ＆ドロップ、またはファイル選択で読み込み、リサイズとJPEG圧縮を行うGUIアプリです。

## 使い方

1. Python (3.8以上) をインストールします。
2. 依存ライブラリをインストールします。
   ```bash
   pip install Pillow tkinterdnd2
   ```
3. アプリを起動します。
   ```bash
   python main.py
   ```

## 機能

- 画像ファイルのドラッグ＆ドロップ読み込み
- 幅・高さ指定でのリサイズ（縦横比維持オプションあり）
- JPEG品質を指定して保存
